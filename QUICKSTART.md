# Быстрый старт

## Запуск проекта

### Вариант 1: Python (рекомендуется)

```bash
# Python 3
python -m http.server 8000

# Откройте в браузере: http://localhost:8000
```

### Вариант 2: Node.js

```bash
# Установите http-server
npm install -g http-server

# Запустите сервер
http-server

# Откройте в браузере указанный адрес
```

### Вариант 3: PHP

```bash
php -S localhost:8000
```

### Вариант 4: VS Code Live Server

Если вы используете VS Code, установите расширение "Live Server" и нажмите "Go Live" в правом нижнем углу.

## Важно!

**НЕ открывайте `index.html` напрямую** (через двойной клик в файловой системе), так как это приведет к ошибкам из-за политики CORS при загрузке модулей.

Всегда используйте локальный веб-сервер!

## Первое использование

1. Откройте сайт через локальный сервер
2. Перейдите на вкладку "Словарь" и изучите несколько слов
3. Попробуйте уроки грамматики
4. Пройдите викторину
5. Проверьте свой прогресс на странице "Прогресс"

## Структура данных

- **Словарь**: `data/words-data.js` - 25 слов в 5 категориях
- **Грамматика**: `data/grammar-data.js` - 6 уроков
- **Викторины**: `data/quiz-data.js` - 10 вопросов

Все данные можно легко расширить, отредактировав соответствующие файлы.
